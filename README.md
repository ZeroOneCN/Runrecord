# 运动步数记录

一个统计个人运动步数的数据化应用，支持步数记录、数据统计、曲线图展示等功能。

## 项目截图

![项目截图](/images/1.png)
![项目截图](/images/2.png)

## 功能特性

- **步数记录管理**：添加、编辑、删除步数记录
- **快速时间选择**：一键选择 9:00、12:00、18:00、23:59 等常用时间点
- **数据统计**：
  - 按天统计：查看每日步数汇总
  - 按月统计：查看每月步数汇总
- **曲线图展示**：直观展示步数变化趋势
- **距离计算**：根据步幅自动计算步行距离（公里）
- **分页功能**：统计数据和记录列表均支持分页
- **用户筛选**：按用户 ID 筛选查看数据

## 技术栈

- **后端**：Node.js + Express
- **数据库**：SQLite
- **前端**：原生 HTML/CSS/JavaScript
- **图表**：Chart.js

## 安装与运行

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
node app.js
```

### 3. 访问应用

打开浏览器访问：http://localhost:9008

## 使用说明

### 添加步数记录

1. 输入用户 ID
2. 输入步数
3. 选择记录时间（可使用快速时间点按钮）
4. 点击"添加记录"

### 查看统计

1. 在"数据统计"区域输入用户 ID
2. 选择月份（每天视图）或年份（每月视图）
3. 可设置步幅（默认 0.7 米）以计算步行距离
4. 曲线图展示步数趋势，卡片显示详细数据

### 管理记录

- 在"所有记录"表格中可编辑或删除已有记录
- 支持按用户 ID 筛选
- 支持分页浏览

## 默认配置

- 服务端口：9008
- 默认步幅：0.7 米
- 每页显示记录数：20 条
- 统计卡片每页显示：12 条
